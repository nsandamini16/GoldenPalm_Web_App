# Golden Palm Resort Website

## Project Overview
Golden Palm Resort Website is a responsive web application designed to present information about a luxury resort. The website provides details about hotel rooms, facilities, services, and contact information for visitors.

This project was developed as a web development project using front-end technologies.

## Live Website
You can view the live website here:
https://nsandamini16.github.io/GoldenPalm_Web_App/

## Features
- Attractive home page with resort introduction
- Navigation bar for easy website navigation
- About Us section describing the resort
- Facilities section showing hotel services
- Guest testimonials section
- Contact information for customers
- Responsive design for different screen sizes

## Technologies Used
The project was built using the following technologies:

- HTML5
- CSS3
- JavaScript
- Bootstrap

## Website Structure
The website includes the following pages:

- Home Page
- Gallery Page
- Packages Page
- Rooms Page
- Contact Page

## How to Run the Project
1. Download or clone the repository from GitHub.
2. Open the project folder.
3. Run the `index.html` file in a web browser.
